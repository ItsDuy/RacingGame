Whether you're mastering the perfect drift or pushing for a new lap record, Racing Simulatordelivers a premium racing experience.

🎮 About the Project

Racing Simulatoris a high-performance local racing game designed for ultra-widescreen and 16:9 displays, developed using Unity 2022.3.44f1. The project focuses on "sim-cade" physics—balancing realistic vehicle dynamics with accessible, fun gameplay.

Key Features:

Advanced Vehicle Physics: Real-time suspension simulation, tire friction modeling, and weight transfer mechanics.

Dynamic Racing Mechanics: Includes professional racing features:

Manual and Automatic Transmission modes.

Drift Scoring System.

Nitro Boost Management.


Precision HUD: Visual indicators for RPM, Speed (KPH/MPH), Gear position, and Best Lap times.

Diverse Track Selection: Race across the racing tracks with different routes.

Customizable Garages: Modify your vehicle's performance specs and aesthetic liveries.

🛠️ Installation Instructions

Follow these steps to get behind the wheel:

Download Unity:

Ensure you have Unity version 2022.3.44f1 (or compatible LTS) installed.

You can manage versions easily via the Unity Hub.

Clone the Repository:

Clone or download the project files from the repository:

git clone [https://github.com/ItsDuy/velocity-drift-3d.git](https://github.com/ItsDuy/velocity-drift-3d.git)


Open the Project:

Launch Unity Hub.

Click on "Add" and navigate to the cloned folder.

Select the project to open it in the Editor.

Build and Run:

Open the Build Settings (File > Build Settings).

Select your target platform (PC, Mac, or Linux).

Click "Build and Run" to generate the executable, or press the Play button in the Editor to start testing immediately.
